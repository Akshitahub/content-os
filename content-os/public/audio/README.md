# Reel background music — PLACEHOLDER ASSETS

`reel-music-1-placeholder-silent.wav`, `reel-music-2-placeholder-silent.wav`,
and `reel-music-3-placeholder-silent.wav` are **genuinely silent, code-generated
WAV files** (see the generator used to create them, described in the
commit that added them). They carry no license claim of any kind because
they contain no actual audio content — they exist only so the Remotion
composition has a valid file to reference for the background-music track.

**These must be replaced with real royalty-free tracks before shipping.**
No royalty-free audio was sourced or downloaded for this task — do not
mistake these for real music, and do not ship them as final assets.

Good sources for real, verifiably royalty-free tracks: Pixabay Music,
YouTube Audio Library, Free Music Archive (CC0/CC-BY tracks only — verify
license text per track before use).
